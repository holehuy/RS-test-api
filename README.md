## Description

A CRUD api using Express Framework and Typescript

## Function

-Create work
-Get list work (with pagination, sort by created day or alphabet)
-Update work title
-Delete work

## Installation

```bash
$ yarn install
```

## Running the app

```bash
#build
$ yarn build
```

```bash
#start
$ yarn start
```

## Running with nodemon

```bash
#development
$ yarn dev
```
